# تشخيص المشاكل - خطوة بخطوة

## الرجاء اتباع هذه الخطوات بالترتيب وإخباري بالنتيجة:

### الخطوة 1: التحقق من تسجيل الدخول
1. افتح التطبيق على http://localhost:3000
2. اضغط F12 لفتح Developer Tools
3. اذهب إلى تبويب **Console**
4. اكتب هذا الأمر واضغط Enter:
   ```javascript
   localStorage.getItem('si_user')
   ```
5. **السؤال**: هل ترى بيانات JSON أم `null`؟
   - إذا كانت `null` → يجب تسجيل الدخول أولاً
   - إذا كانت JSON → انتقل للخطوة 2

---

### الخطوة 2: التحقق من الأخطاء في Console
1. في نفس تبويب Console
2. امسح الشاشة (Clear console)
3. حاول إنشاء بوست جديد
4. **السؤال**: هل ترى أي أخطاء باللون الأحمر؟
   - إذا نعم → انسخ الخطأ بالكامل وأرسله لي
   - إذا لا → انتقل للخطوة 3

---

### الخطوة 3: التحقق من Network Requests
1. اذهب إلى تبويب **Network** في Developer Tools
2. امسح الطلبات (Clear)
3. حاول نشر بوست
4. **السؤال**: هل ترى طلب POST إلى `/api/posts`؟
   - إذا نعم → ما هو Status Code؟ (200, 400, 500؟)
   - إذا لا → لم يتم إرسال الطلب أصلاً

---

### الخطوة 4: وصف المشكلة بالتفصيل
الرجاء الإجابة على هذه الأسئلة:

1. **متى تحدث المشكلة بالضبط؟**
   - [ ] عند فتح صفحة إنشاء البوست
   - [ ] عند الضغط على زر "نشر"
   - [ ] بعد النشر (الصفحة تصبح بيضاء)
   - [ ] عند اختيار Rating Scale
   - [ ] أخرى: __________

2. **ماذا يحدث بالضبط؟**
   - [ ] الصفحة تصبح بيضاء بالكامل
   - [ ] رسالة خطأ تظهر
   - [ ] لا شيء يحدث (الزر لا يستجيب)
   - [ ] أخرى: __________

3. **هل سجلت الدخول بنجاح؟**
   - [ ] نعم
   - [ ] لا
   - [ ] لست متأكد

4. **هل تمكنت من إكمال التسجيل؟**
   - [ ] نعم، التسجيل نجح
   - [ ] لا، لا يزال زر Next لا يعمل
   - [ ] لم أحاول التسجيل

5. **بالنسبة لـ Rating Scale:**
   - [ ] لا تظهر النجوم في صفحة الإنشاء
   - [ ] لا تظهر النجوم بعد النشر في الفيد
   - [ ] النجوم تظهر لكن لا يمكن التفاعل معها
   - [ ] أخرى: __________

## الإجراءات السريعة

### إذا كانت المشكلة: "الصفحة البيضاء"
```javascript
// في Console، اكتب:
console.clear()
// ثم حاول مرة أخرى وانسخ أي خطأ
```

### إذا كانت المشكلة: "زر Next لا يعمل"
```javascript
// في Console، اكتب:
localStorage.clear()
// ثم أعد تحميل الصفحة وحاول التسجيل مرة أخرى
```

### إذا كانت المشكلة: "لا أستطيع تسجيل الدخول"
```javascript
// في Console، اكتب:
fetch('http://localhost:3001/api/users').then(r => r.json()).then(console.log)
// هل ترى قائمة بالمستخدمين؟
```
